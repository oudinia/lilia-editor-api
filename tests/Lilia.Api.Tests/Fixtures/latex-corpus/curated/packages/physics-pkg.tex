\section{Physics package shortcuts}

Derivatives: $\dv{f}{x}$, $\dv[2]{f}{x}$, $\pdv{f}{x}$, $\pdv{f}{x,y}$.

Bra-ket notation: $\ket{\psi}$, $\bra{\phi}$, $\braket{\phi|\psi}$, $\expval{H}{\psi}$.

Absolute value, norm: $\abs{x}$, $\norm{v}$.

Vectors: $\vb{r}$, $\va{v}$, $\vu{e}$.

Identity operator: $\mathbb{1} = \sum_n \dyad{n}{n}$.

Integrals: $\int_0^\infty \!\! e^{-x} \dd{x} = 1$.

Matrix determinant: $\det(A) = \abs{A}$.
