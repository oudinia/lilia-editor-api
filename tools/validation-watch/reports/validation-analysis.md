### Analysis

1. **`Two \documentclass` (55×) — double-wrap.** Almost the entire error volume
   is one document, `d2e72f3f`, whose blocks each already carry a full
   `\documentclass` preamble (an import artifact), so the standalone-validation
   wrapper produces two `\documentclass` lines. Every block in that doc fails
   identically (heading, paragraph, theorem, bibliography all show it). This is
   not user error — the validator should detect a leading `\documentclass` and
   skip/strip its wrapper.

2. **`No counter 'Theorem'` (2×) — env/preamble mismatch.** Theorem blocks store
   `theoremType:"Theorem"` (capitalized) and render `\begin{Theorem}`, but the
   validation preamble never declares `\newtheorem{Theorem}{…}`. Casing matters:
   amsthm convention is lowercase `theorem`. Either normalize `theoremType`
   casing or emit a matching `\newtheorem` for the exact env generated.

3. **Unicode γ/Δ/μ (3×) — literal Greek in prose.** AI-drafted paragraphs put
   literal Greek/relativity notation (`G_μν`, `Δτ`, `γ`) directly in text;
   pdflatex without unicode handling fails. Convert common literal Unicode in
   text to math commands on save/export (extend the existing pass-through /
   font-shim Unicode coverage), or route to a unicode-aware engine.

4. **Listings language (1×).** A code block's `language=` isn't known to
   `listings`. Map editor language ids → listings names; fall back to verbatim
   when unknown rather than erroring.

### ACTION — pipeline fixes, most impactful first

- [ ] **Validator: skip/strip wrapper when block source already has
  `\documentclass`.** Kills ~55 of the recent errors (the entire `d2e72f3f`
  class) and protects every future imported doc.
- [ ] **Theorem exporter: emit `\newtheorem` matching the generated env name, or
  normalize `theoremType` casing** to a known amsthm set.
- [ ] **Unicode coverage: convert literal Greek/super-subscripts in text →
  LaTeX** (extend `SeedPassThroughEnvsAndFontShimCoverage`), or fall back to a
  unicode engine for affected blocks.
- [ ] **Listings: map/whitelist `language=`, verbatim fallback on unknown.**
